The Markdown Structure Create your Markdown file (e.g., _nav.md). Use standard indentation to dictate the first and second levels inside your custom directive block. Markdown :::rcl-header-bar-nav - [Home](/) - [Grooming Services](#) - [Vacaville Grooming Depot](/vacaville) - [Hannah's Pampered Pets](/hannahs) - [Client Galleries](#) - [LD Photography](/ld-photo) - [Contact Us](/contact) ::: 2. The Custom Renderer This static class extracts the ::: block and parses the list into your desired first-level-item and second-level-item hierarchy. It automatically detects nested items and converts the parent into an ARIA-compliant accordion button. The Tag Helper This Tag Helper intercepts the Markdown string, processes it through the renderer, and outputs the raw HTML directly onto the Razor Page without generating an unwanted wrapper tag. C# using Microsoft.AspNetCore.Razor.TagHelpers; using WTS.RazorLibrary.Renderers; namespace WTS.RazorLibrary.TagHelpers { [HtmlTargetElement("markdown-nav")] public class MarkdownNavTagHelper : TagHelper { [HtmlAttributeName("markdown-content")] public string MarkdownContent { get; set; } public override void Process(TagHelperContext context, TagHelperOutput output) { // Strip the tag itself from the DOM output.TagName = null; if (string.IsNullOrWhiteSpace(MarkdownContent)) return; string html = RclNavRenderer.ProcessNavBlocks(MarkdownContent); output.Content.SetHtmlContent(html); } } } 4. 5. Vanilla JavaScript Accordion Logic To make the aria-expanded and hidden attributes functional for the accordion dropdown, add this lightweight script to your site's JS bundle. It ensures screen readers are immediately notified of state changes when the DOM updates. JavaScript document.addEventListener("DOMContentLoaded", () => { document.querySelectorAll('.nav-accordion-toggle').forEach(button => { button.addEventListener('click', () => { // Toggle ARIA state const isExpanded = button.getAttribute('aria-expanded') === 'true'; button.setAttribute('aria-expanded', !isExpanded); // Toggle visibility of the controlled submenu const menuId = button.getAttribute('aria-controls'); const menu = document.getElementById(menuId); if (isExpanded) { menu.setAttribute('hidden', ''); } else { menu.removeAttribute('hidden'); } }); }); });
